# reports

Repositório público criado para armazenar o inventário de relatórios.

## Arquivo principal

O inventário completo deve ser publicado neste repositório como:

```text
inventario_relatorios_campos.md
```

## Observação técnica

O arquivo completo é grande, com aproximadamente 71 MB. Por isso, a forma mais estável de publicação é via Git local, não pelo editor web nem por integração de conteúdo via chat.

## Publicação local recomendada

```bash
git clone https://github.com/mariohcpb/reports.git
cd reports
cp /caminho/do/inventario_relatorios_campos.md ./inventario_relatorios_campos.md
git add inventario_relatorios_campos.md
git commit -m "Adiciona inventario de relatorios"
git push origin main
```

Após o push, o arquivo ficará disponível na raiz do repositório.
